# TimeBank v1.0.0 发布说明

**发布日期:** 2026-03-10  
**版本:** v1.0.0  
**Git 标签:** `v1.0.0`

## 🎯 核心功能

### 1. 积分系统
- ✅ 任务完成获得积分
- ✅ 衰减机制（第 1-2 次 100%，第 3-4 次 75%，第 5+ 次 50%）
- ✅ 完美完成奖励（+2 分）
- ✅ 每日自动重置任务计数

### 2. 兑换系统
- ✅ 短途飞行（10 积分，15 分钟）
- ✅ 国内航线（20 积分，30+2 分钟）
- ✅ 国际航线（30 积分，45+4 分钟）
- ✅ 洲际航线（40 积分，60+8 分钟）⭐ 新增
- ✅ 兑换额度限制（工作日 60 分钟，周末 120 分钟）
- ✅ 额度检查逻辑（只算基础时长，赠送时长不占额度）⭐ 修复

### 3. 计时系统
- ✅ 实时倒计时显示
- ✅ 环形进度条
- ✅ 到站提醒
- ✅ 自动清理过期计时器
- ✅ 按日期分组显示（带分割线）⭐ 新增

### 4. 飞行日志
- ✅ 积分赚取记录
- ✅ 兑换记录
- ✅ 管理员调整记录
- ✅ 按日期分组（带渐变分割线）⭐ 新增
- ✅ 完美完成标记
- ✅ 衰减率显示

### 5. 数据分析 Dashboard
- ✅ 每日积分趋势（折线图）
- ✅ 活跃热力图（按小时 x 星期几）
- ✅ 任务类型分布（饼图）
- ✅ 兑换时间分析（21:30 前后对比）
- ✅ 健康度评估（🟢健康/🟡注意/🔴警示）
- ✅ 日期筛选（7 天/30 天）
- ✅ 周期对比（较上期变化百分比）

### 6. 管理功能
- ✅ 长按设置按钮进入管理面板
- ✅ 任务配置编辑
- ✅ 手动调整余额
- ✅ 数据导出/导入
- ✅ 清除所有数据

## 🔧 技术栈

- **框架:** React 19
- **构建工具:** Vite 6.4.1
- **样式:** Tailwind CSS 4
- **状态管理:** Zustand (带 persist 持久化)
- **图表库:** Recharts 2.15+
- **路由:** React Router DOM 7+

## 🚀 部署说明

### 开发环境
```bash
cd timebank-app
npm install
npm run dev -- --port 5173 --host
```

访问地址：
- 本地：`http://localhost:5173/`
- 网络：`http://<IP 地址>:5173/`

### 生产环境
```bash
npm run build
# 部署 dist 目录到 Web 服务器
```

## ⚠️ 重要修复

### v1.0.0 修复的问题
1. **洲际航线档位缺失** - 添加 40 积分档位（60+8 分钟）
2. **兑换额度统计错误** - 改为只统计 baseMinutes（不含赠送）
3. **兑换检查逻辑不一致** - canRedeem 改为检查 baseMinutes
4. **飞行日志分割线不可见** - 增强为渐变效果 + 蓝色粗体
5. **飞行计时无日期分组** - 添加按日期分组显示
