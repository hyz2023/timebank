## 🔧 技术改动

### engine.js
- ⭐ 新增 `HOLIDAY_2026` 配置对象（包含 workingDaysOff 和 weekendDaysWork）
- ⭐ 新增 `isHoliday(date)` 函数判断是否为法定节假日
- 修改 `getDailyLimit()` 使用 `isHoliday()` 而非简单周末判断
- 限额调整：节假日 90 分钟，工作日 60 分钟

### store.js
- 配置项更新：`dailyExchangeLimitWeekend: 120` → `dailyExchangeLimitHoliday: 90`

### components/RedeemPage.jsx
- 导入 `isHoliday` 函数
- UI 标签从"周末/工作日"改为"节假日/工作日"
- 状态样式根据节假日状态动态变化

## 📝 使用指南

### 查看今日限额
1. 进入"兑换"页面
2. 查看顶部"今日飞行额度"卡片
3. 右上角标签显示"节假日"或"工作日"
4. 右侧显示剩余分钟数 / 总限额

### 节假日自动识别
- 系统自动根据日期判断是否为法定节假日
- 调休上班日自动按工作日限额计算
- 无需手动配置
